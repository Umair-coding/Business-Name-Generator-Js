# 🚀 Business Name Generator

A simple JavaScript project that generates random business names by combining three different categories of words.

## 📖 About

This project creates unique business names by randomly selecting:

- An adjective
- A business category
- A business suffix

Every time the program runs, a new business name is generated.

## ✨ Example Output

```
Card Name is Fire Garments Limited
Card Name is Crazy Foods Bros
Card Name is Amazing Engine Hub
Card Name is Fire Foods Bros
```
